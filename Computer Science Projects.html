Shooter Game

A group projects created with 2 fellow group members uses the SFML library to create a shooter game. In this game, a user must shoot bullets projectiles at various ghosts attacking a tower which attempt to drain the player's health when hitting the tower. The game has both single player and networked multiplayer playing options.

The project was a group effort, but within said effort I worked on audio, projectiles and the money system.

This project can be found here
